Help & Instructions

Welcome to the Face Detection Web App! Follow these steps to use the application effectively:

How to Use:

  • Navigate to the Detect page by clicking "Detect" in the navigation bar.
  • Click the Upload Image button and select an image containing a face.
  • Choose a detection method:
    • MTCNN: Multi-task Cascaded Convolutional Networks for face detection.
    • B-MTCNN: A modified version of MTCNN with improved accuracy.
  • Click the Detect button to process the image.
  • View the detected faces along with confidence scores.

Troubleshooting

  • Image not uploading? Ensure the file is in JPG, PNG, or JPEG format.
  • Faces not detected? Try using a higher-quality image with clear facial features.
  • Slow detection? Large images may take longer to process; try resizing before uploading.
